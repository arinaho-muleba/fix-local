# How to report WATER and SANITATION issues in Joburg

![Placeholder Image](https://storageforfixlocal.blob.core.windows.net/dev/content/aaef8ab6-2f5d-4f2e-b05f-ff81200eb1d7/images/aaef8ab6-2f5d-4f2e-b05f-ff81200eb1d7.webp)

To make sure our city services work well, it's important to report issues like potholes, power cuts, water leaks, or fallen trees. Following up and taking more steps if needed can help fix these issues. Here’s how you can report these issues in Joburg.

Please read point 4 for reporting issues with WATER and SANITATION.

## Steps for reporting issues

1. **Go to the Joburg eservices website to log a fault:**  
- City of Joburg e-services


2. **Call the Joburg Call Centre:**
- Phone: 0860 56 28 74
- Email: joburgconnect@joburg.org.za


3. **Register on Log Now Smart City:**
- Sign up at [Log Now](https://www.lognow.co.za/) to report city problems and see FAQs.


4. **Report WATER and SANITATION issues:**

| **Category**            | **Sub-Category**                                                               | **What You Can Report**                                                                                                      | **Department Contact**                                                                                                        |
|-------------------------|--------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| Water and Sanitation    | Pipes, Sewerage, Streams and Rivers, Toilets in Townships/Informal Settlements  | No water, water outages, burst water pipes and leaks, blocked and overflowing sewers/drains, missing or damaged water valves or meter covers, leaking fire hydrants | **Johannesburg Water**                                                                                                        |
|                         |                                                                                | **Website:** [Joburg Water fault reporting](http://johannesburgwater.co.za.winhost.wa.co.za/ContactUS/contact_us.aspx)       |
|                         |                                                                                | **Phone Number:** 011 688 1400                                                                                                |
|                         |                                                                                | **Email Address:** [customerserviceemails@jwater.co.za](mailto:customerserviceemails@jwater.co.za)                                                                  |
| Refuse Collection       | Waste Management, Wheelie Bins, Garden Sites Management, Public Bins, Rubble Collection, Landfill Site Management | Report issues related to waste management, wheelie bins, garden sites, public bins, rubble collection, and landfill site management | **Pikitup Johannesburg**                                                                                                      |
|                         |                                                                                | **Website:** [Joburg Pikitup waste management reporting](https://www.pikitup.co.za/)                                          |
|                         |                                                                                | **Phone Numbers:** 010 055 5990 OR 087 357 1068                                                                               |
